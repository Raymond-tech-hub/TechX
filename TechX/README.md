# TechX Website

**From Curiosity to Legacy: Pioneering AI Excellence**

This is the official source code for the TechX company website — a modern, performant showcase built with Next.js (App Router), Tailwind CSS, and a focus on highlighting our AI innovations.

TechX is a dynamic young AI and technology company creating intelligent solutions that transform education, automate workflows, and empower global innovation. Our flagship products include **Lumina** (AI-powered adaptive learning platform) and **Nova** (custom LLM AI assistant), alongside services in AI development, automation, game creation, and more.

## 🌟 Features

- Responsive, dark-mode-first design with futuristic accents
- Product spotlights: Lumina & Nova demos
- Sections for Services, About/Team, Contact, and future Blog
- SEO-optimized with Next.js metadata & Open Graph
- Fast performance & accessibility focus
- Easy to customize content via data files

## 🛠 Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript (recommended)
- **Deployment**: Vercel (one-click)
- **Other**: React Server Components, Image optimization, Framer Motion (for animations if added)
